Epic Flow

Visual Kanban for every epic

Track each epic as it moves through the pipeline. The Kanban board gives you a real-time view of agent activity, blockers, and progress across all your projects simultaneously.

  • ✦  Drag epics between stages manually or let the scheduler drive them
  • ✦  Color-coded agent status indicators per card
  • ✦  Inline preview of latest agent output
  • ✦  Filter by project, agent type, or priority
Specialist Roles

A fleet of specialist AI agents

Each agent is a Claude subprocess with a defined role, scoped context, and clear deliverables. Agents collaborate on a shared epic branch without stepping on each other.

  • Architect  — Plans, decomposes, and defines file ownership
  • Implementer  — Writes code from the plan
  • Reviewer  — Audits quality, security, correctness
  • Tester  — Writes tests and validates behaviour
Multi-Project

Manage all your projects in one place

Angy supports multiple repositories and projects simultaneously. Switch between codebases, assign epics per project, and monitor agent activity across your entire portfolio.

  • ✦  Per-project epic queues and history
  • ✦  Independent git branches per epic
  • ✦  Unified dashboard across all repositories
  • ✦  Project-level scheduler configuration
Live Output

Watch agents write code in real time

The Code View streams agent output as it happens — file edits, diffs, shell commands, and test results. Full syntax highlighting powered by JetBrains Mono.

  • ✦  Real-time streaming of agent tool calls
  • ✦  Diff view for every file modified
  • ✦  Shell command output and exit codes
  • ✦  Full session replay per epic
